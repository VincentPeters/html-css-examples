Achtergrond

Met CSS kun je achtergrondkleuren en afbeeldingen instellen.

background-color

background-color: #e3f2fd (lichtblauw)
background-color: #fff3e0 (lichtoranje)
background-color: #e8f5e9 (lichtgroen)
background-color: #f3e5f5 (lichtpaars)
background-color: #333 (donkergrijs, witte tekst) 
div { background-color: #e3f2fd; } /* lichtblauw */
div { background-color: #fff3e0; } /* lichtoranje */
div { background-color: #e8f5e9; } /* lichtgroen */
div { background-color: #f3e5f5; } /* lichtpaars */
div {
  background-color: #333;   /* donkergrijs */
  color: #fff;               /* witte tekst */
}

background-image

Achtergrondafbeelding met background-size: cover 
div {
  background-image: url('../images/voorbeeld-landschap.jpg');
  background-size: cover;      /* vult het hele element */
  background-position: center; /* centreert de afbeelding */
  height: 200px;
}

Gecombineerd: fallback kleur met afbeelding

Als de afbeelding niet laadt, zie je de groene achtergrondkleur 
div {
  /* Fallback kleur - zichtbaar als afbeelding
     niet laadt */
  background-color: #2d6a4f;

  /* Achtergrondafbeelding */
  background-image: url('../images/voorbeeld-landschap.jpg');
  background-size: cover;
  background-position: center;
  height: 200px;
}
