Box Model

Elk HTML-element is een rechthoekige box met content, padding, border en margin.

Box Model Diagram

margin
border
padding
content 
/* Elk element heeft deze lagen: */

/* 1. Content - de inhoud (tekst, afbeelding) */
/* 2. Padding - ruimte rond de content */
/* 3. Border  - rand rond de padding */
/* 4. Margin  - ruimte buiten de border */

div {
  margin: 20px;     /* buitenste laag */
  border: 3px solid black;
  padding: 20px;
  /* content past zich aan */
}

Box Model Properties

Dit is een box met width, padding, border en margin. 
.box {
  width: 250px;           /* breedte van de content */
  padding: 20px;          /* ruimte binnen de border */
  border: 3px solid #1565c0; /* rand */
  margin: 15px 0;         /* ruimte buiten de border */
  background-color: #e3f2fd;
}

Totale Breedte Berekenen

width: 200px
padding: 20px (links + rechts)
border: 2px (links + rechts)
Totaal: 244px
← 244px →

Beide boxes zijn visueel even breed: 200 + 20 + 20 + 2 + 2 = 244px

/* Box A */
.box-a {
  width: 200px;
  padding: 20px;      /* 20 links + 20 rechts */
  border: 2px solid;  /* 2 links + 2 rechts */
}

/* Totale breedte berekening:
   200 (width)
 +  20 (padding-left)
 +  20 (padding-right)
 +   2 (border-left)
 +   2 (border-right)
 = 244px totaal */

/* Box B - zelfde visuele breedte */
.box-b {
  width: 244px;
  padding: 0;
  border: 0;
}
/* Totaal: 244px */

Belangrijk

De totale breedte van een element = width + padding-left + padding-right + border-left + border-right. Zie ook box-sizing: border-box om de berekening eenvoudiger te maken!
