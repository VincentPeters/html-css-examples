Ruimte

Margin en padding bepalen de ruimte rond en binnen elementen.

Padding

Het lichtblauwe gebied toont de padding rond de tekst:

Ik heb padding:
top: 10px | right: 30px
bottom: 20px | left: 40px 
.box {
  padding-top: 10px;
  padding-right: 30px;
  padding-bottom: 20px;
  padding-left: 40px;
  background-color: #0077b6;
  color: #fff;
}

Margin

Het gele gebied is de wrapper; de oranje blokken hebben margin:

margin-bottom: 20px
margin-left: 40px
margin-left: 80px
.box-1 {
  margin-bottom: 20px;
}

.box-2 {
  margin-left: 40px;
  margin-bottom: 20px;
}

.box-3 {
  margin-left: 80px;
}

Meer weten over margin?

W3Schools MDN

Shorthand notatie

Groene achtergrond toont de margin ruimte:

1 waarde

margin: 10px

2 waarden

margin: 10px 30px

4 waarden

margin: 10px 20px 30px 40px
/* 1 waarde: alle zijden gelijk */
margin: 10px;

/* 2 waarden: top/bottom | left/right */
margin: 10px 30px;

/* 4 waarden: top | right | bottom | left */
margin: 10px 20px 30px 40px;

Centreren met margin: 0 auto

Zonder auto margin
margin: 0 auto
/* Element staat standaard links */
.box-links {
  width: 200px;
}

/* Gecentreerd met margin auto */
.box-gecentreerd {
  width: 200px;
  margin: 0 auto;
}
Tip: margin: 0 auto is de eenvoudigste manier om een blok-element horizontaal te centreren. Het element moet wel een vaste breedte (width) hebben.
