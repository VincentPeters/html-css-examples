Typografie

CSS biedt veel mogelijkheden om tekst op te maken.

color

Deze tekst is rood (color: red)

Deze tekst is blauw (color: blue)

Deze tekst is groen (color: green)

Deze tekst gebruikt een hex kleur (color: #e65100)

p { color: red; }
p { color: blue; }
p { color: green; }
p { color: #e65100; }

Meer weten over tekstkleur?

W3Schools - CSS Text

font-family

Serif lettertype (font-family: serif)

Sans-serif lettertype (font-family: sans-serif)

Monospace lettertype (font-family: monospace)

Cursive lettertype (font-family: cursive)

p { font-family: serif; }
p { font-family: sans-serif; }
p { font-family: monospace; }
p { font-family: cursive; }

W3Schools - CSS Fonts

font-size

Tekst op 12px

Tekst op 16px (standaard)

Tekst op 20px

Tekst op 24px

Tekst op 32px

p { font-size: 12px; }
p { font-size: 16px; } /* standaard */
p { font-size: 20px; }
p { font-size: 24px; }
p { font-size: 32px; }

font-weight

font-weight: normal

font-weight: bold

font-weight: lighter

font-weight: 100

font-weight: 400 (= normal)

font-weight: 700 (= bold)

font-weight: 900

p { font-weight: normal; }
p { font-weight: bold; }
p { font-weight: lighter; }
p { font-weight: 100; }
p { font-weight: 400; }  /* = normal */
p { font-weight: 700; }  /* = bold */
p { font-weight: 900; }

text-align

text-align: left (standaard)
text-align: center
text-align: right
text-align: justify - Deze tekst is wat langer zodat je het effect van justify kunt zien. De tekst wordt verdeeld over de volledige breedte van het element.
div { text-align: left; }    /* standaard */
div { text-align: center; }
div { text-align: right; }
div { text-align: justify; }

line-height

line-height: 1.0 (krap)
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.
line-height: 1.6 (comfortabel)
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam.
line-height: 2.5 (ruim)
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam. 
/* Krap - regels zitten dicht op elkaar */
p { line-height: 1.0; }

/* Comfortabel - goed leesbaar */
p { line-height: 1.6; }

/* Ruim - veel ruimte tussen regels */
p { line-height: 2.5; }
