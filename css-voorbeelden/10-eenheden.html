Eenheden

CSS kent verschillende eenheden voor het bepalen van groottes.

Meer info: W3Schools - Units

px (pixels)

Vaste grootte, ongeacht context:

Tekst op 12px Tekst op 16px Tekst op 24px Tekst op 32px 
/* px = absolute eenheid */
.klein  { font-size: 12px; }
.normal { font-size: 16px; }
.groot  { font-size: 24px; }
.groter { font-size: 32px; }

% (percentage)

Relatief aan het ouder-element (blauwe rand):

25%
50%
75%
100%
/* % = relatief aan ouder-element */
.kwart     { width: 25%; }
.half      { width: 50%; }
.driekwart { width: 75%; }
.vol       { width: 100%; }

em (relatief aan ouder)

em stapelt op bij nesting (vermenigvuldigt):

Ouder: 16px (basis font-size: 16px)
Kind: 1.5em = 24px (16px × 1.5 = 24px)
Kleinkind: 1.5em = 36px (24px × 1.5 = 36px) 
/* em = relatief aan ouder font-size */
/* em stapelt op bij nesting! */

.ouder {
  font-size: 16px;
}

.kind {
  font-size: 1.5em; /* 16 × 1.5 = 24px */
}

.kleinkind {
  font-size: 1.5em; /* 24 × 1.5 = 36px */
}

rem (relatief aan root)

rem blijft consistent, ongeacht nesting:

Ouder: 16px (basis font-size: 16px)
Kind: 1.5rem = 24px (root 16px × 1.5 = 24px)
Kleinkind: 1.5rem = 24px (root 16px × 1.5 = 24px, niet 36px!) 
/* rem = relatief aan ROOT font-size */
/* rem stapelt NIET op bij nesting */

.ouder {
  font-size: 16px;
}

.kind {
  font-size: 1.5rem; /* 16 × 1.5 = 24px */
}

.kleinkind {
  font-size: 1.5rem; /* 16 × 1.5 = 24px */
  /* NIET 36px zoals bij em! */
}
Tip: Gebruik rem voor consistente groottes en em wanneer je wilt dat de grootte relatief is aan het ouder-element.
