max-width

width: 100% - Dit element neemt altijd de volledige breedte in.
width: 100% + max-width: 600px - Dit element groeit mee maar wordt nooit breder dan 600px. Verkleint het venster om het effect te zien!

max-width is geweldig voor responsief design! Het element past zich aan op kleine schermen maar wordt niet te breed op grote schermen.

/* Neemt altijd volledige breedte */
.breed {
  width: 100%;
}

/* Groeit mee, maar max 600px breed */
.responsief {
  width: 100%;
  max-width: 600px;
}

/* Tip: max-width is geweldig
   voor responsief design! */
