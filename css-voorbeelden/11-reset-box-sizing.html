/* Browsers voegen standaard stijlen toe:
   - h1-h6 hebben margin boven en onder
   - p heeft margin boven en onder
   - ul/ol hebben margin en padding-left
   - body heeft 8px margin

   Dit verschilt per browser! */

CSS Reset

De meest gebruikte basis reset:

*,
*::before,
*::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Zet dit altijd bovenaan je CSS-bestand.

/* Basis CSS Reset */
*,
*::before,
*::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* * selecteert ALLE elementen */
/* margin: 0 verwijdert standaard marges */
/* padding: 0 verwijdert standaard padding */
/* box-sizing: border-box maakt
   berekeningen makkelijker */

box-sizing vergelijking

Beide boxes hebben width: 200px, padding: 20px en border: 5px:

content-box (standaard)

width: 200px
(alleen inhoud)
Totaal: 250px
5 + 20 + 200 + 20 + 5 = 250px

border-box

width: 200px
(alles inbegrepen)
Totaal: 200px
Padding en border zitten
in de 200px 
/* STANDAARD: content-box */
.box-content {
  box-sizing: content-box;
  width: 200px;
  padding: 20px;
  border: 5px solid #e85d04;
  /* Werkelijke breedte:
     5 + 20 + 200 + 20 + 5 = 250px! */
}

/* BETER: border-box */
.box-border {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid #2d6a4f;
  /* Werkelijke breedte:
     200px (alles inbegrepen) */
}
Tip: Voeg altijd box-sizing: border-box toe aan je CSS reset. Dit zorgt ervoor dat padding en border mee worden gerekend in de breedte.
Let op: Zonder box-sizing: border-box is de werkelijke breedte groter dan je width property!
