CSS Toepassen

Er zijn drie manieren om CSS toe te passen: inline, internal en external.

Inline CSS

Deze tekst is gestyled met inline CSS. De stijl staat direct in het HTML-element.

<p style="color: blue; font-size: 18px;">
  Deze tekst is gestyled met inline CSS.
  De stijl staat direct in het HTML-element.
</p>

Internal CSS

Dit is een heading met internal CSS

<head>
  <style>
    .demo-internal-heading {
      color: #d63384;
      font-size: 1.5rem;
      font-family: Georgia, serif;
      border-bottom: 2px solid #d63384;
      padding-bottom: 0.5rem;
    }
  </style>
</head>

<body>
  <h3 class="demo-internal-heading">
    Dit is een heading met internal CSS
  </h3>
</body>

External CSS

Deze tekst wordt gestyled door een extern CSS-bestand (01-extern-voorbeeld.css).

<!-- In de <head> van je HTML: -->
<link rel="stylesheet" href="01-extern-voorbeeld.css">

<!-- In je HTML body: -->
<p class="demo-extern-tekst">
  Deze tekst wordt gestyled door een extern
  CSS-bestand.
</p>
/* 01-extern-voorbeeld.css */
.demo-extern-tekst {
  color: #2d6a4f;
  font-family: Georgia, serif;
  font-size: 1.2rem;
  border-left: 4px solid #2d6a4f;
  padding-left: 1rem;
}

Belangrijk

External CSS is de aangeraden methode! Zo houd je je HTML en CSS gescheiden. Dit maakt je code overzichtelijker en makkelijker te onderhouden.
