Selectors

CSS selectors bepalen welke HTML-elementen gestyled worden.

Type Selector

Eerste paragraaf - gestyled door de type selector.

Tweede paragraaf - ook gestyled!

Dit is een span, niet een p - dus niet gestyled. 
/* Selecteert ALLE <p> elementen */
p {
  color: #6a1b9a;
  font-size: 1.1rem;
}

Class Selector

Dit is een normale paragraaf.

Dit element heeft de class "highlight".

Nog een normale paragraaf.

Dit element heeft ook de class "highlight".

/* Selecteert elementen met class="highlight" */
.highlight {
  background-color: yellow;
  padding: 0.2rem 0.4rem;
}
<p>Dit is een normale paragraaf.</p>
<p class="highlight">Dit element heeft de class "highlight".</p>
<p>Nog een normale paragraaf.</p>
<p class="highlight">Dit element heeft ook de class "highlight".</p>

ID Selector

Dit is een gewone paragraaf.

Dit element heeft id="speciaal".

Dit is weer een gewone paragraaf.

/* Selecteert het element met id="speciaal" */
#speciaal {
  color: #e65100;
  font-weight: bold;
  font-size: 1.3rem;
  border: 2px dashed #e65100;
  padding: 0.5rem;
}
<p>Dit is een gewone paragraaf.</p>
<p id="speciaal">Dit element heeft id="speciaal".</p>
<p>Dit is weer een gewone paragraaf.</p>

Descendant Selector

Deze paragraaf staat buiten het article - niet gestyled.

Deze paragraaf staat BINNEN het article - wel gestyled!

Deze ook - want ook binnen het article.

Deze paragraaf staat weer buiten het article - niet gestyled.

/* Selecteert alleen <p> binnen <article> */
article p {
  color: #1565c0;
  font-style: italic;
  border-left: 3px solid #1565c0;
  padding-left: 0.5rem;
}
<p>Buiten het article - niet gestyled.</p>
<article>
  <p>Binnen het article - wel gestyled!</p>
  <p>Deze ook - want ook binnen het article.</p>
</article>
<p>Weer buiten het article - niet gestyled.</p>

Belangrijk

Vermijd het overmatig gebruik van ID selectors. Classes zijn flexibeler omdat ze herbruikbaar zijn. Een ID mag maar één keer voorkomen per pagina, terwijl een class op meerdere elementen kan worden toegepast.
