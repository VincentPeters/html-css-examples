Hexadecimaal

#ff0000
#00ff00
#0000ff
#333333
#f00
/* Hex: # + 6 tekens (rr gg bb) */
.rood    { color: #ff0000; }
.groen   { color: #00ff00; }
.blauw   { color: #0000ff; }
.donker  { color: #333333; }

/* Shorthand: 3 tekens als paren gelijk */
.rood    { color: #f00; }
/* #f00 = #ff0000 */

RGB

rgb(255, 0, 0)
rgb(0, 128, 0)
rgb(0, 0, 255)
/* RGB: waarden van 0 tot 255 */
.rood  { color: rgb(255, 0, 0); }
.groen { color: rgb(0, 128, 0); }
.blauw { color: rgb(0, 0, 255); }

Meer weten over RGB kleuren?

W3Schools MDN

RGBA (transparantie)

Het gestreepte patroon toont de transparantie:

alpha: 1.0 volledig
alpha: 0.7
alpha: 0.4
alpha: 0.1 bijna transparant 
/* RGBA: rgb + alpha (0.0 - 1.0) */
.vol      { background: rgba(0, 0, 255, 1.0); }
.licht    { background: rgba(0, 0, 255, 0.7); }
.lichter  { background: rgba(0, 0, 255, 0.4); }
.bijna    { background: rgba(0, 0, 255, 0.1); }

/* alpha: 0.0 = volledig transparant */
/* alpha: 1.0 = volledig dekkend */
