# HTML & CSS Voorbeelden

Praktische voorbeelden bij de cursus **HTML & CSS: De Basis van het Web** voor IoT-studenten aan KdG Hogeschool.

## Inhoud

- **HTML Voorbeelden** - Basis template, koppen, tekst, lijsten, links, afbeeldingen, semantische elementen
- **CSS Voorbeelden** - Selectors, box model, typografie, kleuren, eenheden, en meer
- **Portfolio** - Een compleet portfolioproject als eindoefening

## Bekijken

Open `index.html` in je browser of bekijk de site via GitHub Pages.
